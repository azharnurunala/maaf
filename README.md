# Apology Ebook — Landing Page

Landing page permohonan maaf Nurunala × Hujan Pustaka atas keterlambatan
pengiriman buku *Ternyata Kita Belum Siap Nikah* & *Belajar Mencintai*,
sekaligus halaman unduh dua ebook gratis (Menulis Novel & Menulis Memoar).

Mobile-first, animasi hujan & buku mengambang, palet terakota (Clay).

## Cara pakai (GitHub Pages)

1. Upload semua isi folder ini ke repository.
2. Settings → Pages → Source: `main` / root.
3. Buka URL Pages-nya. `index.html` tampil otomatis.

`index.html` sudah **self-contained** (semua gambar, font, dan runtime
ter-embed) — file ini saja cukup untuk live. File lain hanya sumber.

## Isi folder

| File | Keterangan |
|------|-----------|
| `index.html` | Halaman jadi, standalone — yang dilihat pengunjung |
| `Apology Ebook.dc.html` | Sumber yang bisa diedit (butuh `support.js` + cover) |
| `support.js` | Runtime untuk membuka file sumber |
| `cover-*.png` | Gambar cover buku & ebook |
